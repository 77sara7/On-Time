iMacros.Component Namespace

The iMacros.Component namespace contains the iMacrosControl for creating applications that take full advantage of the features available in the iMacros Browser (http://www.imacros.net/). The iMacrosControl provides a tabbed web browser (as in the iMacros Browser) and exposes methods which give access to iMacros player full capabilities.
