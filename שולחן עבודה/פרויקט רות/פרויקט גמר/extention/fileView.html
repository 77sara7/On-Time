The new Chrome version no longer allows extensions to access the file system. So by default iMacros stores the macros inside bookmarks.

If you want to read and write macros as text files from/to your hard drive, please download the ontime for Chrome File Access Installer .

The file access module is a small Windows, Mac, and Linux application that allows ontime Chrome to save files and share them with its ontime Internet Explorer and ontime Firefox siblings.
